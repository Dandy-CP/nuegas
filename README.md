# Nuegas
This Project for Portfolio Purpose

[Demo Website](https://nuegas-beta.vercel.app/)

## Installation

Install Project with npm:

```bash
  npm install
```

Run the development server:

```bash
  npm run dev
```

## Tech Stack
- Typescript
- Next.Js
- TailwindCSS
- MaterialUI
- React Query
- Socket.io
